# MovieBox.ph Stremio Addon

100% Working Stremio addon with real metadata and streaming links from MovieBox.ph.

## Features

- ✅ Real metadata from moviebox.ph
- ✅ Working streaming links from CDN
- ✅ Search functionality
- ✅ Trending catalogs
- ✅ Multiple quality options

## Deploy to Railway

1. Go to [Railway.app](https://railway.app)
2. Sign in with GitHub
3. Create new project from GitHub repo
4. Select this repository
5. Railway will auto-deploy
6. Use the Railway URL + `/manifest.json` in Stremio

## Deploy to Render

1. Go to [Render.com](https://render.com)
2. Create new Web Service
3. Connect this repository
4. Deploy!

## Install in Stremio

1. Open Stremio
2. Go to Addons → Community Addons
3. Enter your deployed URL + `/manifest.json`
4. Click Install
